#+title: org-roam-cjk

org-roam-cjkは、CJK言語で[[https://github.com/org-roam/org-roam][org-roam]]を使用するときの補助機能を追加するパッケージです。

現時点では日本語用プラグインのみ存在します。これはorg-roamバッファーにおける「リンクされていない参照（unlinked references）」をリスト表示するセクションに日本語で書かれた文章を対応させるものです。デフォルトでorg-roamに含まれている実装では、ripgrepの検索パターン用正規表現に単語境界（\b）が使われています。そのため、日本語を含めたCJK言語のような境界文字に頼らない言語で書かれた文章に対しては、正しい検索が行われません。このような点について修正が加えられています。

インストレーションの詳細については英語版README（README.org）を参考にしてください。
